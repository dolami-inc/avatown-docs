## Prerequisites

### Unity

互換性を確保するために、[公式Unityウェブサイト](https://unity.com/releases/editor/whats-new/2022.3.22)から`Unity 2022.3.22f1`をダウンロードしてインストールすることを推奨します。このバージョンは、VRChatで使用されているものと一致しています。

**重要**:
VRChatの最新SDKにはUnity 2022が必要です。現在もUnity 2019を使用しているユーザーは、早急にアップグレードすることを強く推奨します。今後のSDKアップデートは古いバージョンでは利用できなくなり、また、旧バージョンで作成されたコンテンツは互換性の問題が発生する可能性があります。

### VRchat Creator Companion

VRChatのプロジェクトを設定するには、VRChat Creator Companion が必要です。インストール手順については、[公式VRChatドキュメント](https://vcc.docs.vrchat.com) をご参照ください。

### Avatown Unity クライアント

対応するアバターのUnitypackageおよびアセットバンドルをアップロードするには、Avatown Unity クライアント が必要です。まずは、お好みの言語に応じたAvatown Unity クライアント をダウンロードしてください。

- [Avatown Unity クライアント（英語版）](https://github.com/dolami-inc/auto-uploader-docs/releases/download/Beta/AvatownUnityClient_v1.0.1.unitypackage)
- [Avatown Unity クライアント（日本語版）- 近日公開予定](#)

ダウンロード後、Unityパッケージを任意のディレクトリに保存してください。

### ステップ 1: 新規プロジェクトの作成
新しくプロジェクトを開始する場合は、以下の手順に従ってください。
既存のプロジェクトにAvatown Unity クライアント をインポートする場合は、ステップ 2に進んでください。

VRChat Creator Companion を開き、新しいプロジェクトを作成してください。

<img height="200" noZoom src="/images/unityclient/vrc_create.png" />

プロジェクト名を入力し、ディレクトリを指定した後、「Create Project」 をクリックしてください。
次の画面では、必要なVRChat SDKがすべて選択されていることを確認してください。

<img height="200" noZoom src="/images/unityclient/vrc_package.png" />

必要なパッケージを選択したら、「Open Project」 をクリックしてUnityを起動してください。

### ステップ 2: Avatown Unity クライアント のインポート
プロジェクトがUnityで開かれ、使用可能な状態になったら、Avatown Unity クライアント をインポートします。
すでにアバターのプロジェクトをお持ちの方は、ここからの手順に従ってください。

インポートするには、Assets > Import Package > Custom Package... を選択してください。

<img height="200" noZoom src="/images/unityclient/import_asset.png" />

先ほどダウンロードした Avatown Unity クライアント の Unity パッケージファイルを選択してください。

<img height="200" noZoom src="/images/unityclient/select_package.png" />

Unityにインポートされるファイルの一覧が表示されるプロンプトが出ます。「Import」 をクリックしてください。

<img height="200" noZoom src="/images/unityclient/click_import.png" />

インポートされるファイルの一覧が表示されるプロンプトが表示されます。「Import」 をクリックして進めてください。
Unityがインポート処理を開始するので、完了するまでお待ちください。

<img height="200" noZoom src="/images/unityclient/select_package.png" />

### ステップ 3: Avatown Unity クライアント へのアクセス
インポートが正常に完了すると、UnityのツールバーにAvatownが表示されます。
「Avatown クライアント」 をクリックしてクライアントインターフェースを起動してください。
ログイン画面が表示されるので、goavatown.com のアカウント資格情報を使用してログインしてください。
ログインが完了すると、Avatown Unity クライアント を使用できるようになります。

<img height="200" noZoom src="/images/unityclient/avatown_login.png" />

ログインすると、Avatown で作成した商品の一覧が表示されます。

<img height="200" noZoom src="/images/unityclient/product_list.png" />

商品の詳細を確認するには、商品カードの右上にある「View」ボタン をクリックしてください。
これにより、該当商品の詳細情報が表示されます。

<img height="200" noZoom src="/images/unityclient/upload_unitypack.png" />

既存のアバタープロジェクトに Avatown クライアント をインポートしている場合、「Upload UnityPackage」 をクリックすると、現在のシーン内で VRC Avatar Descriptor がアタッチされている利用可能なアバターの一覧がドロップダウンに表示されます。
以下の画像は、現在のシーンにある VRC Avatar Descriptor が設定されたアバターの例を示しています。

<img height="200" noZoom src="/images/unityclient/selected_project.png" />

次に、「Select this Avatar」 ボタンをクリックしてください。
システムがビルドの問題をチェックし、問題がなければアバターのアップロードが開始されます。

<img height="200" noZoom src="/images/unityclient/confirm.png" />

次に、アセットバンドルのアップロード も同様の手順で行ってください。
Unitypackage と アセットバンドル のアップロードが完了すると、アバターが自動アップロード対応である ことが表示されます。

<img height="200" noZoom src="/images/unityclient/confirm.png" />

以上で完了です！🎉
あなたのアバターはユーザーが利用できる状態になりました。
最後までお読みいただき、ありがとうございました！
